# SISY.cam - Video Recording Tool for Women Experts

Live site: https://sisy.cam

Part of the [SISY ecosystem](https://sisy.world) - visibility infrastructure for women experts.

## About

Browser-based video recording tool. Record a professional speaker intro in 2 minutes with guided prompts, auto-transcription, and AI enhancements.
